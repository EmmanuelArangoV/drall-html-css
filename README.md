## Installation

### Prerequisites

- Git installed on your machine
- A text editor (VS Code recommended)
- A modern web browser

### Steps

1. **Clone the repository**

```bash
git clone https://github.com/EmmanuelArangoV/drall-html-css.git
cd drall-html-css
```

2. **Open with Live Server**

Open `index.html` with Live Server extension in VS Code, or simply double-click the file to open it in your browser. 

3. **Explore the project**

Browse through the responsive design and test the interactive navigation menu. 

## Usage

### Viewing the Landing Page

Simply open the `index.html` file in any modern web browser.  The page is fully functional without any build process or server setup.

### Customizing Content

#### Changing Colors

Edit the gradient colors in `style.css`:

```css
background: linear-gradient(135deg, #ff8f70 0%, #ff3d54 100%);
```

#### Modifying Text

Update the content directly in `index.html`:

```html
<h1>A modern publishing platform</h1>
<p>Grow your audience and build your online brand</p>
```

#### Adding New Sections

Follow the existing HTML structure and add corresponding CSS styles: 

```html
<section class="new-section">
  <h2>Your New Section Title</h2>
  <p>Your content here</p>
</section>
```
